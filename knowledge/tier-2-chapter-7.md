# Chapter 7: TrackWrestling Setup

## The Backbone of Modern Tournaments

TrackWrestling isn't optional anymore — it's essential. Without it, you're running blind.

Here's what TrackWrestling does:
- Manages brackets automatically
- Provides live results for families
- Handles seeding correctly
- Tracks tournament statistics
- Professionalizes your event

This chapter gets your TrackWrestling ready BEFORE event day.

---

## Account Setup (First Time)

### Creating Your Organization

1. Go to trackwrestling.com
2. Create organization account
3. Add your logo/branding
4. Set up admin users (limit to 2-3 people)

### Configuration Settings

Set these defaults:
- Default scoring (NFHS)
- Match length by age division
- Point values (standard)
- Near fall values

---

## Creating Your Event

### 4 Weeks Before Event

1. **Create new event**
   - Organization → Events → Create
   - Fill in basic info (name, date, venue)
   
2. **Add registration**
   - Set dates (early bird, regular, day-of)
   - Set pricing
   - Enable online registration

3. **Configure divisions**
   - Add age groups (6U, 8U, 10U, 12U, 15U, HS)
   - Add weight classes
   - Add girls divisions if needed

### Common Mistakes to Avoid

| Mistake | Impact | Prevention |
|---------|--------|------------|
| Forgetting to add division | Wrestlers can't register | Double-check all ages |
| Wrong weight classes | Wrestling wrong weights | Verify against state rules |
| Wrong birthdate ranges | Ineligible wrestlers compete | Verify cutoffs |
| Seeding criteria not set | Wrong seeding | Set before registration closes |

## Seeding Configuration

### Setting Up Seeding

In TrackWrestling:
- Go to Event → Seeding
- Set criteria (State Champ, State Placer, etc.)
- Add verification requirements

### Standard Seeding Criteria

| Place | Requirement |
|-------|-------------|
| #1 Seed | State Champion |
| #2 Seed | State Placer (2nd-4th) |
| #3 Seed | Region Champion |
| #4 Seed | Region Placer |
| #5-8 Seed | Place at Qualifier or League Champ |

### Seeding Rules to Communicate

From Tim's actual event instructions:

> "No bracket adjustments will be considered for seeding purposes. Seeding based on criteria entered when registering. Criteria must be verifiable. Any wrestler found to be falsifying seeding criteria will be removed."

## Bracket Management

### Bracket Formats

| Format | When to Use | Notes |
|--------|-------------|-------|
| Double Elimination | Standard | Winners bracket + consi |
| Round Robin | Small field (6 or less) | Everyone wrestles everyone |
| Round Robin + Placement | Medium field | Top 4 place after RR |

### Bracket Generation

- Generate brackets AFTER registration closes
- Generate 24 hours before event
- Export to PDF as backup
- Check for errors BEFORE posting

## Live Results Setup

### The Spectator Feature

TrackWrestling's live results let families see:
- When their kid is wrestling
- Mat assignment
- Match results
- Bracket updates

### Setup Steps

1. Enable "Live Results" in event settings
2. Get the live results URL
3. Test BEFORE event day
4. Share URL in pre-event communications

## Technical Backup

### What Could Go Wrong?

- Computer crashes
- Internet/WiFi fails
- TrackWrestling server issues
- User error

### Backup Procedures

| Scenario | Solution |
|----------|----------|
| Computer crash | Backup laptop ready |
| Internet fails | Offline mode (download brackets) |
| Server issues | Print brackets, manual scoring |
| User error | Admin access, undo function |

**Pro tip:** Always print bracket sheets before event. Store in folder for backup.

## TrackWrestling Best Practices

### For Tournament Directors

1. **Don't wait until last minute** — Set up 4 weeks early
2. **Test everything** — Run a mock event if needed
3. **Have backup people** — Know who can access system
4. **Print backups** — Paper doesn't crash
5. **Communicate URL** — Parents need live results link

### For Table Workers

- Always click "Save Result"
- Never use browser back button
- If mistake made, notify head table immediately
- Use correct bout numbers
