## The On-Time Tournament System

### 5 Steps to Events That Start On Time — Guaranteed

---

### Step 1: Work Backward from Start Time

Your tournament starts at 10:00 AM. Here's the formula:

| Time | Task |
|------|------|
| 10:00 AM | First match on ALL mats |
| 9:45 AM | Mats cleared, final announcements |
| 9:30 AM | First call for division |
| 9:15 AM | Weigh-in CLOSED for first division |
| 9:00 AM | Final weigh-ins completed |
| 8:30 AM | Doors open, weigh-ins begin |
| 8:00 AM | Staff arrives, equipment check |
| 7:30 AM | Venue setup complete |

**The Rule:** If weigh-ins run late, push back start time by the exact minutes lost. **Never start late.**

---

### Step 2: The Mat Flow Principle

Empty mat = management failure.

Every mat should have a match within 2 minutes of the previous match concluding.

**How to do it:**
- Use a "next match ready" system
- Runner calls next two wrestlers to staging area
- Mat-side official signals "mat clear" only after match complete

This is the **trust metric** — if your mats are empty, something's wrong.

---

### Step 3: The Communication Timeline

Parents don't ask questions when they know the answers.

Send a "Pre-Event Email" with:
- Venue address + directions
- Weigh-in windows (specific times)
- Exact start time (emphasize ON TIME)
- What to bring

Post the schedule on social media. Prepare "day-of" update templates.

**Pro tip:** Tell them what they need to know *before* they need to know it.

---

### Step 4: Staffing That Works

- 1 certified ref per mat
- 1 scorer + 1 runner per mat
- 2 weigh-in staff minimum
- 1 floor manager for real-time issues

Don't skimp. Enough staff = fast event. Not enough staff = chaos.

---

### Step 5: TrackWrestling Setup (Before Event Day)

Your tech should be ready before you walk in the door.

- Create event in TrackWrestling 2 weeks out
- Input all divisions, weight classes, bracket formats
- Upload wrestler registrations
- Test live results URL
- Have backup laptops charged

Tech failure isn't an excuse. Plan for it.
