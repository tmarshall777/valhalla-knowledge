## Recommended: Browse.ai

**Why it's best for you:**
- **Easy to train** — You show it the event page once, it learns
- **Handles dynamic content** — Modern websites like TrackWrestling use JavaScript
- **Scheduled runs** — Can run automatically on your schedule
- **No coding required** — Point-and-click interface

**How it works:**
1. You give it a URL (e.g., event results page)
2. You show it what data to extract (name the elements)
3. It scrapes and outputs to Excel/CSV/JSON
4. You can schedule it to run weekly

**Pricing:**
- Free tier: Limited
- Starter: ~$30/mo
- Pro: ~$60/mo

---

## Alternative: Octoparse

**Why it's powerful:**
- Specifically designed for login-protected sites
- Can handle "infinite scroll" and dynamic loading
- More powerful than Browse for complex sites

**Pricing:**
- Standard: $75/mo
- Professional: $199/mo

---

## Alternative: Make.com + ScraperAPI

**Why it's flexible:**
- Make (formerly Integromat) is a workflow tool
- Can combine with ScraperAPI for anti-bot handling
- Can send data anywhere (Google Sheets, email, etc.)

**Pricing:**
- Make: $9-600/mo (based on operations)
- ScraperAPI: $49+/mo

## The Solution with Browse.ai (Simplest)

1. **Sign up** for Browse.ai
2. **Create a bot** for TrackWrestling:
   - Give it your event URL (from your account)
   - Show it what to click/navigate
   - Tell it what data to extract (names, placements, weights)
3. **Set schedule** — Run every Sunday after events
4. **Output** — Gets delivered to Google Sheet or CSV

**Then I can:**
- Pull that data
- Build rankings
- Create newsletter content
- Generate social posts
