# Chapter 10: The Day Before

## Setup Day

The day before your event is when you set the foundation for success. Don't leave for event day wondering "did we forget something?"

---

## The Day Before Timeline

### Morning (9 AM - 12 PM)

1. **Final headcount check**
   - Pull registration report from TrackWrestling
   - Note any last-minute additions/drops
   - Update brackets if needed

2. **Equipment audit**
   - Pack van with all gear
   - Check items against packing list
   - Verify tech (laptops, cords, batteries)

### Afternoon (12 PM - 5 PM)

3. **Venue setup (if same-day setup possible)**
   - Arrive at venue
   - Set up mats
   - Set up table area
   - Set up registration desk
   - Test WiFi and tech

4. **Bracket finalization**
   - Make final adjustments
   - Generate final brackets
   - Print copies (2-3 sets)
   - Post one set for public viewing

5. **Staff prep**
   - Print staff schedules
   - Prepare staff credentials/wristbands
   - Brief floor manager (call or meeting)

### Evening (5 PM - 8 PM)

6. **Weigh-in session (if offered)**
   - Staff weigh-in station
   - Process wrestlers through
   - Record weights in system
   - Issue wristbands

7. **Final walkthrough**
   - Walk the venue
   - Visualize the flow
   - Check PA system
   - Confirm signage in place

8. **Early rest**
   - Get sleep!
   - Don't stay up late fixing things
   - Trust your preparation

## Setup Checklist (Venue)

### Before Leaving for Venue

- [ ] Laptops charged + backup
- [ ] Extension cords
- [ ] Printer + ink + paper
- [ ] Bracket sheets (printed)
- [ ] Clipboards + pens
- [ ] Table covers
- [ ] Mat tape
- [ ] Signage (printed)
- [ ] First aid kit
- [ ] Staff food/snacks
- [ ] Water for staff

### At Venue - Setup Tasks

- [ ] Unload equipment
- [ ] Position mats (correct spacing)
- [ ] Set up table area
- [ ] Set up registration desk
- [ ] Set up awards area
- [ ] Test PA system
- [ ] Test WiFi connection
- [ ] Set up scoreboard/computers
- [ ] Place signage
- [ ] Final walkthrough

---

## Day Before Staff Actions

### For Referees
- Confirm arrival time
- Send venue address + parking info
- Remind to bring food/water

### For Table Workers
- Confirm assignment (which mat)
- Confirm arrival time
- Remind training session 8:30 AM

### For Floor Manager
- Final briefing
- Review any last-minute changes
- Confirm communication plan

---

## What NOT to Do Day Before

| Don't | Why |
|-------|-----|
| Stay up late | You'll be exhausted day-of |
| Panic about minor issues | Fix what matters, let go rest |
| Make major changes | Trust your prep |
| Argue with staff | Keep peace |
| Skip meals | You need energy tomorrow |

---

## Morning-Of Checklist Reminder

For event day morning, you've already prepared:

- [ ] Staff knows arrival time
- [ ] Venue ready (if set up previous day)
- [ ] Brackets printed
- [ ] Tech tested
- [ ] You've slept
