## Phase 1: Pre-Event Planning (2 Weeks Out)

### Venue & Logistics

- [ ] Confirm venue booking and get final headcount
- [ ] Map out floor plan — mark mat locations, table areas, weigh-in station
- [ ] Calculate mat requirements: 1 mat per 8-10 wrestlers for efficiency
- [ ] Confirm HVAC — venue must be temperature-controlled (wrestlers cut weight in heat)
- [ ] Verify adequate parking and ingress/egress for 500+ attendees
- [ ] Confirm WiFi availability (critical for TrackWrestling live results)

### Staffing

- [ ] Recruit certified officials: 1 ref per mat, minimum
- [ ] Schedule table workers: 1 scorer + 1 runner per mat
- [ ] Assign weigh-in staff (2 minimum)
- [ ] Designate "floor manager" to handle real-time issues
- [ ] Confirm vendor contacts for emergency equipment backup

### Technology

- [ ] Set up TrackWrestling account and create event
- [ ] Input all divisions, weight classes, and bracket formats
- [ ] Upload wrestler registrations and verify weights
- [ ] Test live results URL — ensure spectator access works
- [ ] Have backup laptops and power strips on-site

### Communication

- [ ] Send "Pre-Event Email" to all registered families:
  - Venue address + directions
  - Weigh-in windows
  - Schedule (specific start time)
  - What to bring
- [ ] Post schedule on social media with clear start time
- [ ] Prepare "Day-of" updates template for any last-minute changes

## Phase 2: The Day Before

### Setup Preparation

- [ ] Print bracket sheets (backup if tech fails)
- [ ] Pre-assign mat positions to divisions
- [ ] Create "Mat Assignment Sheet" for officials and runners
- [ ] Prepare weigh-in station: certified scale, table, forms
- [ ] Charge all laptops, score clocks, and backup batteries

### Staff Briefing

- [ ] Send final schedule to all staff with arrival time
- [ ] Confirm official count for each mat assignment
- [ ] Brief lead official on "special circumstances" (disputed matches, weight issues)

### Contingency Planning

- [ ] Identify backup venue options (if catastrophic failure)
- [ ] Have vendor contact list accessible
- [ ] Prepare "emergency delay" script for PA system

## Phase 3: Event Day — Backward Planning

### The Golden Rule: Work Backward from Start Time

Your tournament start time is **10:00 AM**. Use this formula:

| Time | Task |
|------|------|
| 10:00 AM | First match on ALL mats |
| 9:45 AM | Mats cleared, final announcements |
| 9:30 AM | First call for division |
| 9:15 AM | Weigh-in CLOSED for first division |
| 9:00 AM | Final weigh-ins completed |
| 8:30 AM | Doors open, weigh-ins begin |
| 8:00 AM | Staff arrives, equipment check |
| 7:30 AM | Venue setup complete |

**Rule**: If weigh-ins run late, push back start time by the exact minutes lost. DO NOT start late.

### Check-In Flow

- [ ] Open doors at 8:30 AM
- [ ] Weigh-in station opens immediately
- [ ] Assign "weigh-in greeter" to direct traffic
- [ ] Mark wristbands on verified wrestlers
- [ ] NO exceptions to weigh-in windows — late = not competing

### Bracket Management

- [ ] Generate "mat cards" for each wrestler (division, weight, mat #)
- [ ] Post all brackets on wall with clear mat assignments
- [ ] Runners receive copy of all mat cards
- [ ] Floor manager has master bracket with live status

## Phase 4: Running the Event

### Mat Flow Principles

- [ ] NO mat sits empty during competition — if a match finishes, next match called immediately
- [ ] "Trust metric": Empty mat = management failure. Every mat should have a match within 2 minutes of previous conclusion.
- [ ] Use "next match ready" system: Runner calls next two wrestlers to staging area
- [ ] Mat-side official signals "mat clear" only after match complete

### Communication During Event

- [ ] PA announcements every 30 minutes with schedule updates
- [ ] Floor manager updates bracket sheets in real-time
- [ ] If delay exceeds 15 minutes, notify families via text/email
- [ ] Social media person posts "rounds in progress" updates

### Problem Resolution

- [ ] Weight disputes: Have certified scale available for re-weigh at request
- [ ] No-show: Call name 3 times, then default after 2 minutes
- [ ] Bracket errors: Floor manager resolves, NOT table workers
- [ ] Technical difficulties: Backup laptop ready, IT contact on speed dial

## Phase 5: Post-Event

### Immediate

- [ ] Confirm all final results are entered in TrackWrestling
- [ ] Post results to website and social media
- [ ] Send "Thank You" email with next event date

### Within 48 Hours

- [ ] Distribute staff payments
- [ ] Send venue feedback survey to host
- [ ] Collect "lessons learned" from staff
- [ ] Note what worked and what needs adjustment

### Within 1 Week

- [ ] Update operations guide with improvements
- [ ] Send survey to families (what did we do well, what needs work)
- [ ] Begin planning for next event using this same checklist
