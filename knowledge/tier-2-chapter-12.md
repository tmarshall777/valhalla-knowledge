# Chapter 12: Running the Event

## The Running System

Now the event is live. Your job is to keep it running smoothly. Here's how.

---

## The Core Rules During Event

### Rule #1: Mats Always Active
- Empty mat = management failure
- If a match ends, next match starts within 2 minutes
- Runner must have next wrestlers ready

### Rule #2: Communication Constant
- PA announcements every 30 minutes
- Floor manager updates brackets in real-time
- Any delays communicated immediately

### Rule #3: Problems Get Escalated
- Table worker → Floor Manager → TD
- Don't let table workers make big decisions
- Floor manager has authority to solve most issues

---

## Match Management

### The Flow

```
Mat finishes
    ↓
Runner calls next wrestlers (from staging)
    ↓
Wrestlers report to mat
    ↓
Referee calls match
    ↓
Match completes
    ↓
Next wrestlers called
```

### Runner Protocol

From Tim's instructions:
> "Mat Chat when you need a wrestler or coach"

Use Mat Chat (TrackWrestling feature) or walkie talkies to communicate:
- "Mat 3, bring your next wrestlers"
- "Mat 2, we have a scratch"

---

## Session Transitions

### Between Sessions

| Time | Action |
|------|--------|
| T - 30 min | Announce final matches in session |
| T - 15 min | Final call for session |
| T - 0 min | Session ends, awards begin |
| T + 30 min | Awards complete |
| T + 45 min | Next session weigh-ins begin |

### Multi-Session Management

- Keep sessions tight (don't run over)
- Awards ceremonies between sessions
- Clear the floor for next session
- Don't let morning session push into afternoon

---

## Communication Templates (PA)

### Announcements to Have Ready

**Opening:**
> "Welcome to [Event Name]. Our first session begins at [Time]. Good luck to all wrestlers!"

**Mid-Event:**
> "[Division] matches are now on mat. Good luck to all competitors."

**Session Change:**
> "The [Morning/Afternoon] session has concluded. Awards are in [Location]. The [Afternoon/Evening] session begins at [Time]."

**Final Call:**
> "This is a final call for [Division]. Please report to staging. Wrestling begins in 15 minutes."

---

## Managing the Brackets

### What Can Go Wrong

- Wrong wrestler called
- Weight class mismatch
- Seed error discovered
- Forgot to scratch a no-show

### Solutions

- Use track numbers (red/green) to identify
- Verify before every match
- Floor manager resolves
- No adjustments after match starts

| Task | How |
|------|-----|

| Handle issues | Floor manager escalation |
