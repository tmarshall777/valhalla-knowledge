# Chapter 9: The Week Before

## The Final Push

One week out — this is when preparation becomes execution. Get this right, and event day runs smooth.

---

## 1 Week Out Checklist

### Staffing Confirmation
- [ ] Confirm all refs are coming
- [ ] Confirm table workers
- [ ] Confirm floor manager
- [ ] Send schedule to all staff
- [ ] Confirm backup people

### Technical Preparation
- [ ] Generate brackets in TrackWrestling
- [ ] Verify bracket accuracy
- [ ] Print bracket sheets (backup)
- [ ] Test live results URL
- [ ] Charge all laptops
- [ ] Test WiFi at venue

### Communication
- [ ] Send pre-event email with full schedule
- [ ] Post on social media
- [ ] Update website with final details
- [ ] Send weigh-in reminders

### Logistics
- [ ] Confirm venue access
- [ ] Confirm vendor deliveries
- [ ] Order awards (if not done)
- [ ] Arrange food for staff
- [ ] Print all forms/checklists

### Emergency Preparedness
- [ ] Emergency contact list ready
- [ ] First aid kit stocked
- [ ] EMT/Trainer confirmed
- [ ] Backup computer ready

## Final Pre-Event Checks

### Bracket Verification

Before calling brackets final, verify:
- [ ] All registered wrestlers in correct division
- [ ] Seedings applied correctly
- [ ] Weight classes balanced
- [ ] Bye placement correct
- [ ] Seeding criteria verified

### Technical Verification

- [ ] TrackWrestling event page live
- [ ] Live results URL working
- [ ] Printouts made
- [ ] Backup system ready
- [ ] WiFi login obtained

### Communication Verification

- [ ] Email sent with schedule
- [ ] Social posts published
- [ ] Website updated
- [ ] FAQs answered

---

## The Mindset This Week

**Trust, but verify.**

Assume nothing. Check everything. It's easier to fix a problem now than deal with it on event day.
