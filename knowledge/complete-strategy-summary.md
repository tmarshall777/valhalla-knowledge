## Marketing & Content Strategy

### Newsletter Plan
- **Platform:** Beehiiv (recommended)
- **Frequency:** 2x/month (off-season), increase during season
- **Content Mix:**
  - 50% Tips & value
  - 25% Services/Courses
  - 15% Personal/Behind the scenes
  - 10% Event promotions (season)

### Email Sequence Created (4 Emails)
1. "The mistake that makes every tournament run late" → Lead magnet download
2. "Here's the system that fixes it" → Tier 2 preview
3. "One TD transformed his event using this method" → Consulting case study
4. "If you don't have time to plan..." → Tournament management services

### Social Media Strategy
- **Frequency:** 3-4 hours/week (batch record Mon, edit Tue, post Wed/Fri)
- **Content Pillars:**
  - Behind the Scenes
  - Pro Tips
  - Valhalla Insider
- **Tools:** Canva AI voice + CapCut for editing (automated workflow)
- **Alternative:** HeyGen AI avatar (more expensive, less manual)

### Marketing Hooks Developed (Top 7)
1. Stat hook: #1 complaint late starts, #2 events too long
2. Question: Why do some start on time while others run late?
3. Authority: 100+ events, 20+ years
4. Secret reveal: Giving system away for free
5. Promise: Start on time, end in 3 hours
6. Differentiator: Which one are you running?
7. Reverse psychology: Don't need more mats, need system
