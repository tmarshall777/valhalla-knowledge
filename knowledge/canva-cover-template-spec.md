## Canva Template Link (Create This)

In Canva, search for:
- "ebook" or "book cover" templates
- Or create custom with dimensions 8.5" x 11"

---

## Quick Steps in Canva

1. **Create Design** → Custom Size → 8.5" x 11"
2. **Add Background** → Color fill (Navy Blue #1a365d)
3. **Add Text** → Use Georgia or similar serif font
4. **Add Divider** → Gold line (#d69e2e)
5. **Save Template** → Use for all chapters
