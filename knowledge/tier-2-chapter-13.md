# Chapter 13: Problem Resolution

## When Things Go Wrong (Because They Will)

Even the best-planned events have problems. The difference between amateur and professional tournaments is how problems are handled.

This chapter gives you protocols for the most common issues.

---

## Problem Categories

### 1. Weight Disputes
**Scenario:** Parent claims wrestler is at correct weight but scale shows otherwise.

**Solution:**
- Have certified scale for verification
- Allow one re-weigh in singlet (no clothes)
- If dispute persists, floor manager decides
- Document all disputes

### 2. Bracket Errors
**Scenario:** Wrestler in wrong bracket, seeding wrong, or missing from bracket.

**Solution:**
- Floor manager reviews
- If before match: fix and re-print
- If after match: match stands, fix for future rounds
- Document all errors

### 3. No-Shows
**Scenario:** Wrestler doesn't report to mat when called.

**Solution:**
- Call name 3 times
- Wait 2 minutes
- Default the match
- Document for bracket purposes

### 4. Referee Issues
**Scenario:** Referee calls in sick, or makes controversial call.

**Solution:**
- Have backup ref on call
- Floor manager discusses with ref
- If call wrong, may discuss (but officials' calls final)
- Never argue with refs publicly

### 5. Technical Failures
**Scenario:** Computer crashes, WiFi fails, TrackWrestling freezes.

**Solution:**
- Switch to paper brackets
- Continue matches manually
- Have backup laptop ready
- Never stop event for tech issues

### 6. Parent/Coach Complaints
**Scenario:** Someone complains about seeding, match outcome, or scheduling.

**Solution:**
- Listen calmly
- Don't argue
- Floor manager handles
- If escalated, TD meets privately
- Never argue in public

## The Escalation Framework

| Level | Who Handles | Examples |
|-------|-------------|----------|
| **1** | Table Worker | Minor score corrections |
| **2** | Floor Manager | Bracket errors, disputes |
| **3** | TD | Major issues, refunds, venue |

---

## Emergency Protocols

### Medical Emergency
- Stop the match
- Clear the mat
- Call for EMT/Trainer
- If serious, call 911
- Document everything

### Facility Emergency
- Fire: Evacuate per venue plan
- Power outage: Continue with manual scoring
- HVAC failure: Assess safety, decide continue/cancel

### Security Issue
- Contact venue security
- If serious, call 911
- Document everything

---

## Problem Resolution Best Practices

### Do
- Stay calm
- Listen first
- Solve problems privately
- Document issues
- Learn from mistakes

### Don't
- Don't argue publicly
- Don't make exceptions for anyone
- Don't panic
- Don't ignore problems
- Don't blame staff publicly

---

## Post-Event Review

Within 48 hours of EVERY event:

1. **What went right?**
2. **What went wrong?**
3. **What would we change?**
4. **What did we learn?**

Write it down. Update your playbook.
